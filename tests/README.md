# SmartRisk CZ5 — Sprint 3.1

Versión `3.1.0-alpha`.

## Dashboard Ejecutivo

Este sprint corrige la ruta `/dashboard` e incorpora un Centro de Operaciones con:

- KPIs interactivos.
- Alertas automáticas.
- Situación provincial.
- Próximos vencimientos.
- Estado ENOS.
- Calidad de datos.
- Actividad reciente.
- Accesos rápidos.
- Exportación CSV.
- Lectura de datos locales y sincronizados.

## Prueba local

Ejecute `frontend/index.html` con Live Server y abra:

`http://127.0.0.1:5500/frontend/index.html#/dashboard`

No utilice el `index.html` antiguo de la raíz del repositorio.
