# R02.3 — Brechas operativas y flujo UGR–Coordinación

## Incorporado

- Motor de valoración de calidad operativa con diez criterios.
- Clasificación automática: operativamente viable, requiere ajustes, brecha importante y no ejecutable.
- Panel territorial de brechas y agenda al 23 de julio de 2026.
- Contador dinámico de días hasta la actualización.
- Compromisos por acción para la siguiente actualización.
- Registro de recursos, producto esperado, indicador, articulación y solicitud de asistencia.
- Verificables documentales, fotográficos, geográficos o mixtos.
- Estados de revisión: Borrador, Enviado a revisión, Observado, Corregido y Validado.
- Bandeja de Coordinación para novedades, solicitudes y acciones pendientes de revisión.
- Historial local de cambios, usuario y fecha de actualización.
- Navegación directa desde una brecha territorial hacia la ficha de la acción.

## Alcance técnico

La persistencia de este incremento continúa en `localStorage`. El flujo ya es funcional en una misma instalación y deja preparado el modelo de datos para la posterior sincronización multiusuario.
