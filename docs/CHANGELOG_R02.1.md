# Changelog R02.1

- Se crea `SmartRisk.Workspace` para persistir provincia y cantón.
- Se transforma Territorios en un workspace territorial.
- Se agregan selectores de provincia y cantón.
- Se integran KPIs de plan, sitios, acciones e instituciones.
- Se agregan alertas de sitios prioritarios y acciones bloqueadas.
- Se incorporan accesos contextualizados a los demás módulos.
- Se añade exportación CSV del resumen cantonal.
- Se mantiene compatibilidad con datos mock, Google Sheets y Kobo.
