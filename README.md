# SmartRisk CZ5 — Sprint 1.7B

Versión `v2.9.0-alpha`.

## Integración ArcGIS Online

El módulo incorpora:

- Portal y Item ID configurables.
- Elemento inicial: `444ae547115c4e4c941eae219af4841d`.
- Consulta de metadatos mediante ArcGIS REST API.
- Detección del tipo de elemento.
- Carga de Web Maps.
- Carga de Feature Layers y Feature Services.
- Apertura embebida de aplicaciones compatibles.
- Leyenda, búsqueda, galería de mapas base y listado de capas.
- Coordenadas en tiempo real y escala métrica.
- Incorporación temporal de sitios SmartRisk con coordenadas.
- Apertura directa del elemento en ArcGIS Online.
- Soporte opcional para token.

## Seguridad

No se deben publicar tokens institucionales dentro del repositorio. Para la versión productiva se recomienda OAuth 2.0 o un backend intermediario.

## Ejecución

Use GitHub Pages o un servidor local. La integración depende de recursos externos de ArcGIS y requiere conexión a internet.
