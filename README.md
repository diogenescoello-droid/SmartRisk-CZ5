# SmartRisk CZ5 — MVP v1.0

MVP web autónomo para demostración y entrega inicial.

## Ejecutar
Abra `index.html` en Chrome, Edge o Firefox.

## Acceso
- Usuario: `admin@smartrisk.cz5`
- Contraseña: `SmartRisk2026!`

## Incluye
- Login demostrativo.
- Dashboard.
- Navegación por Usuarios, Territorios, Instituciones, Sitios críticos y Acciones.
- Alta y edición de registros.
- Persistencia local mediante `localStorage`.
- 48 territorios cargados.
- Resumen de 48 usuarios: 47 activos y 1 pendiente.

## Alcance técnico
Esta versión sirve para demostración funcional. La autenticación es local y no debe considerarse seguridad de producción.
La conexión de escritura con Google Sheets requiere un backend o Apps Script con credenciales protegidas.

## Publicación rápida
Puede publicarse como sitio estático en GitHub Pages:
1. Crear repositorio.
2. Subir estos archivos.
3. En Settings > Pages, seleccionar la rama principal y carpeta raíz.
